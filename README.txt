ICC Men's T20 World Cup – Master Data
=====================================

All datasets in this folder are generated by code in source/data_creation.ipynb.
The pipeline is still in development (work in progress).


Project structure
-----------------

  Sports_analytics/
    master_data/           <-- you are here
      README.txt           (this file)
      Years/               Raw match JSONs from Cricsheet (one file per match)
        {year}/
          {match_id}.json
      dataset/             Generated CSVs (output of source/data_creation.ipynb)
        *.csv
    source/
      data_creation.ipynb  Code that builds all dataset/*.csv from Years/*.json


Data source
-----------

Raw files: Cricsheet (https://cricsheet.org). ICC Men's T20 World Cup matches
(T20, male). JSON format v1.0.0 / 1.1.0. Some matches withheld (Afghanistan
policy); see cricsheet.org/withheld-matches.

Years included: 2016, 2021, 2022, 2024.
Example: Years/2024/1415708.json = Ireland vs India (2024-06-05).


Folder: dataset/
----------------

All files below are generated by source/data_creation.ipynb. Re-run the
notebook to regenerate after adding or changing JSONs in Years/.

  master_matches.csv
    One row per match. Columns: match_id, date, season, city, venue, team1,
    team2, toss_winner, toss_decision, winner, result_type, result_margin,
    event_name, event_match_number, event_group, player_of_match, overs,
    balls_per_over, gender, match_type, data_version.

  master_deliveries.csv
    One row per ball. Columns: match_id, inning_no, batting_team, bowling_team,
    over, ball_in_over, batter, bowler, non_striker, runs_batter, runs_extras,
    runs_total, is_wicket, wicket_player_out, wicket_kind, wicket_fielder,
    extras_type.

  master_deliveries_with_commentary.csv
    Same as master_deliveries plus commentary_short and commentary. Commentary
    is filled only for 2024 matches where a text commentary sheet was available
    (e.g. Ireland vs India, India vs Pakistan).

  players_info_teams.csv
    One row per player per match. Columns: match_id, date, season, event_name,
    team, player_name, player_id (Cricsheet registry), primary_role
    (batter/bowler/allrounder from delivery counts), is_player_of_match.

  batting_stats.csv
    One row per player (career in this dataset). Columns: player_name, matches,
    innings, not_outs, runs, balls_faced, average, strike_rate, hundreds,
    fifties, fours, sixes. Balls faced exclude wides.

  bowling_stats.csv
    One row per player. Columns: player_name, matches, overs, balls_bowled,
    runs_conceded, wickets, average, economy, strike_rate, four_wickets,
    five_wickets. Run outs not credited to bowler.

  player_stats.csv
    One row per player: batting and bowling stats merged. Columns include
    matches_bat, innings, runs, bat_avg, bat_sr, matches_bowl, overs, wickets,
    bowl_avg, economy, bowl_sr, four_wickets, five_wickets.
